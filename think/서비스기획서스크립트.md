# 중고거래 플랫폼 서비스 기획서 스크립트

```
Title: CarrotSafe - 안전한 중고거래 플랫폼 기획서

1. 고객군
- 목표고객군: 안전한 중고거래를 원하는 취약계층 (여성, 고령자, 주부 등)
- Early Adopter: 중고거래 경험이 있는 20-40대 여성 주부층

2. 문제
- 현재문제:
  - 거래상대방 신원확인의 어려움 (중요도 4.4점)
  - 직거래 시 안전 불안감 (특히 여성 사용자)
  - 기술 취약계층의 앱 접근성 문제

- Existing Alternative:
  - 당근마켓/번개장터 (기본 매칭만 제공)
  - 오프라인 중고매장 (선택권 제한, 가격 경쟁력 부족)
  - 가족/친구 동행 (임시방편, 근본적 해결책 아님)

3. 고유가치제안
- 고유가치제안: "취약계층도 안심하고 참여할 수 있는 신뢰 중심의 포용적 중고거래 생태계"
- High-level concept: "모든 세대가 안심하는 중고거래, CarrotSafe"

4. 솔루션
- 직관적 안전 표시 시스템
- 모바일 우선 간편 인터페이스
- 실시간 채팅 강화
- 거래 패턴 분석 대시보드

5. 경쟁우위
- 언더서비드 마켓(취약계층) 공략 전략
- 포용적 접근성 기술 및 안전 중심 아키텍처
- 커뮤니티 기반 신뢰 시스템
- 취약계층 특화 데이터 및 안전도 평가 알고리즘

6. 수익구조
- 프리미엄 안전 서비스 구독(월 9,900원)
- 안전 거래 수수료(거래액의 1-2%)
- 신뢰도 인증 서비스(건당 19,900원)
- 안전 관련 파트너십 수익

7. 비용구조
- 고정비: 개발팀 인건비, 서버 인프라, 사무실 운영비
- 변동비: 고객 획득 비용, 안전 서비스 운영비, 결제 수수료
- 초기 1년 예상 비용: 월 32,000,000원 + 변동비

8. 핵심지표
- Acquisition: 월간 신규 가입자 수, 취약계층 가입 비율
- Activation: 첫 거래 완료율, 안전 기능 활용률
- Retention: 월간 리텐션율(목표 60%), 재거래율
- Revenue: 프리미엄 전환율, ARPU, LTV/CAC
- Referral: 추천 가입 수, NPS

9. 채널
- 모바일 앱(iOS/Android)
- 웹 서비스
- 커뮤니티 마케팅(네이버 카페, 맘카페)
- 오프라인 교육 프로그램 연계
- 시니어/여성 단체 파트너십
```
